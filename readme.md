# 🚀 Nexus Support - Plataforma de Gestión de Servicios IT

![Angular](https://img.shields.io/badge/Angular-17+-DD0031?style=for-the-badge&logo=angular&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?style=for-the-badge&logo=typescript&logoColor=white)
![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)

**Nexus Support** es una aplicación web moderna diseñada para la visualización y gestión de servicios tecnológicos. Permite a los usuarios explorar un catálogo de soluciones que van desde soporte técnico 24/7 hasta implementaciones avanzadas de Inteligencia Artificial y Ciberseguridad.

---

## 📋 Características del Sistema

- **Catálogo Dinámico:** Visualización de servicios con filtrado en tiempo real.
- **Detalle de Servicios:** Información expandida, precios y características específicas de cada solución.
- **Sistema de Favoritos:** Persistencia local para marcar y guardar servicios de interés.
- **Arquitectura Robusta:** Gestión de datos centralizada a través de Servicios de Angular.
- **Persistencia:** Uso de `LocalStorage` para mantener los datos del usuario entre sesiones.
- **Diseño Responsivo:** Interfaz optimizada para dispositivos móviles y escritorio.

## 🛠️ Tecnologías Utilizadas

- **Angular 17+**: Framework principal para la construcción de la SPA (Single Page Application).
- **TypeScript**: Lenguaje de programación para un desarrollo tipado y seguro.
- **CSS3**: Estilos personalizados con variables para una gestión de diseño limpia.
- **Vercel**: Plataforma elegida para el despliegue continuo y hosting de alto rendimiento.

## 🚀 Instalación y Uso

Sigue estos pasos para ejecutar el proyecto de forma local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/nexus-support.git
   cd nexus-support/nexus-support-app
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el servidor de desarrollo:**
   ```bash
   ng serve
   ```

4. **Ver el proyecto:**
   Abre tu navegador en `http://localhost:4200/`
